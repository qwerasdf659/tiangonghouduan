# 推广合作协议（模板）

> ⚠️ 重要声明：本模板为商业合作框架参考，**非法律意见**。正式签署前必须经本地劳动法/合同法律师审阅，尤其"双方关系性质"与"防假外包"相关条款。模板中【】为待填写项。

> 适用：早期地推（纯佣金合伙人，无底薪、无社保，平台与地推为平等合作关系）
> 平台方：天工商户营销平台（以下简称"甲方"）
> 数字依据：操盘手已定提成方案（首单奖 100 元/出单新店；扶店分成=该店佣金 12%、前 3 个月；续约标准 8 万有效流水）

---

## 协议正文

**甲方（平台方）**：【公司/经营主体全称】　统一社会信用代码：【】　地址：【】
**乙方（合作推广方）**：【姓名】　身份证号：【】　联系方式：【】

甲乙双方本着平等、自愿、诚信原则，就乙方为甲方平台拓展合作商户事宜，达成如下协议：

### 第一条　合作关系性质（核心条款）

1.1 双方系**平等民事合作关系，非劳动关系、非雇佣关系**。乙方不受甲方考勤、坐班、排班管理，自主安排推广的时间、地点与方式。

1.2 甲方不向乙方支付固定工资、不缴纳社会保险及住房公积金；乙方收入为按本协议约定的推广佣金。

1.3 乙方自行承担其经营/劳务所得对应的个人所得税及相关税费。

### 第二条　合作内容

2.1 乙方负责向线下商户推广甲方平台，促成商户入驻并开通消费返积分等合作功能。

2.2 乙方拓展的商户须经甲方审核通过、正式入驻并产生真实消费交易，方计入乙方业绩。

### 第三条　佣金与结算

3.1 **无底薪**。乙方收入由以下两部分构成：

- **首单奖**：乙方拓展的新商户，在入驻当月产生首笔真实消费（经甲方审核通过）的，甲方给予乙方 **100 元/店** 的一次性拉新奖励。
- **扶店分成**：乙方拓展的新商户，自入驻起 **前 3 个自然月**，甲方按该商户每月产生的"平台佣金"的 **12%** 支付乙方分成；自第 4 个月起，该商户转由甲方维护，乙方不再就该商户获得分成。

3.2 **计算基数**：分成基数为"平台佣金"（=商户消费流水 × 平台与商户约定的佣金比例），**非消费流水（GMV）本身**。

3.3 **结算口径**：佣金以甲方平台后台"审核通过"的真实消费数据为准。被风控判定为刷单、虚构、异常的交易不计入，已结算的予以扣回。

3.4 **结算周期**：按自然月结算，次月【】日前支付上月佣金。乙方应配合提供合规的收款凭证（个人经营所得/劳务报酬开票等）。

### 第四条　业绩与续约标准

4.1 乙方合作首月（自【】起 25 个工作日内）应促成不低于 **8 万元有效消费流水**（有效=审核通过、无刷单）。

4.2 首月未达 4.1 标准的，甲方有权**不再续约**（合作自然终止），已产生的合法佣金照常结算。此为合作续约条件，非劳动关系下的解除。

### 第五条　真实性与数据归属

5.1 乙方不得以虚构交易、诱导刷单等方式骗取佣金；违反者甲方有权扣回相应佣金并立即终止合作，情节严重的依法追责。

5.2 乙方在合作中拓展的商户信息、用户数据、交易数据均归甲方所有，乙方不得擅自留存、披露或用于其他用途。

### 第六条　保密与竞业

6.1 乙方对合作中知悉的甲方商业信息（佣金政策、商户名录、平台数据等）负保密义务，合作终止后仍然有效。

6.2 合作期内，乙方不得为与甲方构成直接竞争的平台拓展同一商户（具体范围由双方另行约定）。

### 第七条　合作期限与解除

7.1 本协议自双方签字（盖章）之日起生效，有效期【】个月，期满未续签自动终止。

7.2 任何一方提前 **【7】日**书面通知对方，可解除本协议；乙方存在第五条违规情形的，甲方可立即解除且无需补偿。

7.3 协议解除/终止时，甲方结清乙方已产生的合法佣金；乙方应交接商户关系与资料。

### 第八条　税务与合规

8.1 本协议项下佣金均为税前金额，乙方自行申报并承担个人所得税及相关税费。

8.2 双方确认本协议为合作/劳务关系，乙方不主张劳动关系项下的工资、社保、经济补偿等权利。

### 第九条　争议解决

9.1 因本协议产生的争议，双方协商解决；协商不成的，提交甲方所在地【人民法院/仲裁机构】解决。

### 第十条　其他

10.1 本协议未尽事宜，双方可签订补充协议，与本协议具同等效力。

10.2 本协议一式两份，甲乙双方各执一份，自签署之日起生效。

## 附：签署前必做的合规检查（防"假外包真用工"）

以下任一情形可能导致被认定为劳动关系（须补社保+赔偿），签署及履行中务必避免：

| 红线 | 要求 |
| --- | --- |
| ❌ 考勤打卡 | 不得要求乙方坐班、打卡、固定上下班 |
| ❌ "工资"字样 | 结算单写"佣金/服务费"，不写"工资/薪金" |
| ❌ 工牌/排班/罚款 | 不得用员工化管理（工牌、排班表、罚款制度） |
| ❌ 强制培训考核 | 不得像管员工一样强制考勤式培训、绩效考核 |
| ✅ 自主展业 | 乙方自主决定何时、何地、如何推广 |
| ✅ 合规结算 | 走劳务报酬/个人经营所得，不进工资表、不代扣社保 |
