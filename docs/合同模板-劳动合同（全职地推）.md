# 劳动合同（模板·全职地推转正用）

> ⚠️ 重要声明：本模板为参考框架，**非法律意见**。劳动合同受《劳动合同法》强约束，正式使用前必须经本地劳动法律师审阅，尤其"试用期""录用条件""提成结算"条款。【】为待填写项。

> 适用：地推跑出业绩后转全职（含底薪、社保、合法试用期 + 阶梯提成）
> 数字依据：东莞 2026 口径（最低工资 2080、社保最低基数 4775、单位社保约 1026/月）+ 操盘手提成方案

---

## 合同正文

**甲方（用人单位）**：【公司全称】　统一社会信用代码：【】　地址：【】　法定代表人：【】
**乙方（劳动者）**：【姓名】　身份证号：【】　住址：【】　联系方式：【】

依据《中华人民共和国劳动合同法》等法律法规，甲乙双方平等自愿协商一致，签订本合同。

### 第一条　合同期限与试用期

1.1 本合同为固定期限劳动合同，期限 **1 年**：自【】年【】月【】日起至【】年【】月【】日止。

1.2 试用期 **2 个月**：自【】年【】月【】日起至【】年【】月【】日止。
（合规说明：合同期 1 年以上不满 3 年，试用期依法最长 2 个月；本合同满足。）

### 第二条　录用条件（试用期考核依据，关键条款）

2.1 乙方试用期内须达成以下录用条件，**乙方确认入职前已知悉并认可**：

- 自入职起 **25 个工作日内**，促成不低于 **8 万元有效消费流水**（有效=平台审核通过、无刷单/虚构）。

2.2 试用期内未达成 2.1 录用条件的，甲方有权依据《劳动合同法》第三十九条"试用期被证明不符合录用条件"解除合同，并以平台后台审核通过的业绩数据为证据。

### 第三条　工作内容与地点

3.1 乙方担任【地推专员】岗位，负责拓展商户入驻平台、维护商户关系等。

3.2 工作地点：东莞市【】及甲方业务覆盖区域。

### 第四条　工作时间

4.1 执行标准工时制，每日 8 小时、每周 40 小时（或依甲方依法制定的考勤制度）。

### 第五条　劳动报酬

5.1 **底薪**：乙方月底薪为人民币【2080】元（不低于东莞市现行最低工资标准 2080 元/月；为减少社保基数倒挂争议，建议设为接近社保基数的金额，由甲方定）。

5.2 **试用期工资**：不低于正式底薪的 80%，且不低于最低工资 2080 元。

5.3 **提成（阶梯制，转正后适用）**：以乙方拓展商户产生的"平台佣金"为基数，按月：
- 当月对应佣金达保本线以下：仅发底薪；
- 超出部分按【8%~15%】阶梯计提（具体档位由甲方薪酬制度规定，所有岗位提成总额受 GMV 2% 团队预算约束）。

5.4 提成以平台"审核通过"的真实消费数据结算；刷单/虚构交易不计且扣回。

5.5 工资于次月【】日前以银行转账支付。

### 第六条　社会保险与公积金

6.1 甲方依法为乙方缴纳养老、医疗、失业、工伤、生育五项社会保险，个人部分由甲方代扣代缴。

6.2 住房公积金按【国家及东莞市规定/双方约定比例】缴存。

6.3 乙方知悉：社保个人部分按社保缴费基数（下限 4775 元）依法代扣，扣后实发可能低于应发底薪，此为法定代扣，不属于克扣工资。

### 第七条　保密与竞业限制

7.1 乙方对甲方商户名录、佣金政策、平台数据等商业秘密负保密义务。

7.2 如约定竞业限制，甲方应在离职后竞业限制期内依法支付经济补偿，具体另行签订竞业限制协议。

### 第八条　合同解除与终止

8.1 双方依《劳动合同法》约定解除、终止本合同。

8.2 试用期内，乙方不符合第二条录用条件的，甲方可依法解除；甲方须留存业绩数据等证据。

8.3 合同解除/终止时，甲方依法结清工资、提成，办理离职交接与社保转移。

### 第九条　争议解决

9.1 因本合同发生争议，可向甲方所在地劳动争议仲裁委员会申请仲裁。

### 第十条　其他

10.1 本合同一式两份，双方各执一份，自双方签字（盖章）之日起生效。

10.2 未尽事宜依国家及东莞市相关法律法规执行。

## 附：使用要点

| 要点 | 说明 |
| --- | --- |
| 试用期合法前提 | 合同签 1 年以上才能设 2 个月试用期；录用条件须**入职前书面告知并签字**，否则试用期解除无效 |
| 录用条件要可量化 | "25个工作日 8万有效流水"必须写进合同/录用确认书，平台数据是证据 |
| 工资与社保基数 | 底薪 2080 但社保按 4775 基数缴，实发约 1665；建议把底薪提到接近基数以减少纠纷 |
| 提成受预算约束 | 所有岗位提成总额锁定 ≤ GMV 2% |
| 何时用本合同 | 地推已稳定跑出 8万+/月流水、值得转全职时；早期请用《推广合作协议》（纯佣金） |
