### 5.1 服务性质

道具商城是平台提供的**官方向用户单向售卖虚拟道具**的服务（B2C 单向）。所有购买以虚拟资产（星石）结算，**不涉及任何法定货币**，**不存在用户之间的交易**。

### 5.2 购买规则

**（一）** 道具由官方统一上架并定价，用户使用星石向官方购买。**用户不可自行挂单出售物品，用户之间不进行任何交易撮合。**

**（二）** 道具**买入即消耗**：购买后即归用户使用，**不可卖回平台、不可退回星石、不可转让给其他用户**（平台不提供任何回收/回购通道）。

**（三）** 道具商城仅售卖**纯虚拟道具**（如补签卡、置顶道具、对话框皮肤等系统外无变现价值的道具）。实物、优惠券及任何具有人民币参考价的物品**不进入道具商城**。

**（四）** 星石仅可用于购买零价值虚拟道具、参与官方竞价、兑换广告曝光，**不可用于购买实物或优惠券**（实物仅可通过水晶系资产在兑换商城兑换）。

### 5.3 官方竞价（可选玩法）

平台可对部分官方道具发起竞价活动，由用户使用星石出价、价高者得。中标用户支付的星石被消耗（销毁），未中标用户的冻结星石**全额解冻退回**（退还未支出的星石，非回购）。竞价仅使用星石结算，不构成用户间交易。

### 5.4 禁止行为

用户不得从事以下行为：

## 第七章 消费回馈活动规则

> 本章所述"消费回馈活动"，是平台基于用户消费情况向用户发放虚拟资产回馈的促销互动服务。回馈在用户的消费回馈额度内动态随机发放，**每次参与均必得回馈，不存在"未获得"的情况**，不设固定中奖概率。

### 7.1 活动性质

**（一）** 平台提供的消费回馈活动为**消费促销与趣味互动服务**：用户到合作商户消费累计获得积分，使用积分参与回馈，获得不同数量的虚拟资产回馈好礼。

**（二）** 本活动属"消费附赠/回馈"性质，**积分仅能通过线下消费返还获得，不可使用人民币或任何法定货币购买**，平台不提供任何"付费参与""购买回馈机会"的通道。

### 7.2 活动公示信息

平台在回馈活动页面公示以下信息，公示内容真实有效、不得擅自变更（有利于用户的除外）：

| 公示项 | 内容 |
|--------|------|
| 活动名称 | 餐厅消费回馈 |
| 主办方 | 东莞市墨珩数字科技有限公司 |
| 联系方式 | 平台内"意见反馈"功能或在线客服 |
| 活动时间 | 以活动页面公示的起止时间为准 |
| 参与条件 | 持有平台积分达到活动公示的门槛（如积分≥100分） |
| 参与方式 | 消费返积分，使用积分参与回馈，每次消耗一定积分 |
| 参与次数 | 以活动页面公示的每日上限为准 |
| 回馈好礼 | 各档位包含的虚拟资产好礼名称与数量（见 7.3） |
| 发放机制 | 动态随机发放、每次必得（见 7.4） |
| 好礼到账 | 回馈好礼直接发放至用户账户余额 |

### 7.3 回馈好礼设置

**（一）** 回馈活动的好礼**均为虚拟资产**（如星石、各色源晶碎片、积分等不同数量的组合），**不含实物大奖，不标注人民币价格**。

**（二）** 回馈好礼按价值分为高、中、低三档及保底档，各档包含的好礼名称与数量以活动页面公示为准。

**（三）** **每次参与均可获得回馈好礼，不存在"未获得"的情况**；当较高价值档位暂不可得时，将发放低档或保底好礼，保证每次必得。

**（四）** 回馈好礼均为平台内虚拟资产，单项好礼的价值远低于法定上限，本活动最高奖价值不超过人民币五万元。

### 7.4 发放机制说明（动态发放，不设固定概率）

**（一）** 本活动采用**动态发放机制**：用户每次参与可得的回馈档位，由系统根据用户的**累计消费额度、回馈资格额度、近期获得情况、保底进度**等多重因素实时综合决定。

**（二）** 由于可得档位随上述因素动态浮动、且因人而异、因时而异，**本活动不存在固定不变的单一中奖概率数值，平台据实不公示固定概率**，改以本条如实说明发放机制。该机制对所有用户依统一规则运行。

**（三）** 发放规则要点：

- **消费越多，越有机会获得高档回馈**：用户消费积累的回馈资格额度越高，可解锁的好礼档位越高；额度不足时，较高价值档位可能暂时无法获得，此时将获得低档或保底好礼。
- **每次必得、保底兜底**：系统设有保底机制，保证每次参与均有回馈好礼产出，不存在空手而归的情况。
- **连续未得高档的补偿**：当用户连续多次未获得较高价值好礼时，后续获得较高价值好礼的机会将自动提升。

### 7.5 公平性声明

- 回馈结果由系统按统一规则的随机算法生成，**平台不对任何特定用户的单次回馈结果进行人为暗箱干预**（不存在"指定某用户必得/必不得某好礼"的操作）。
- 发放机制的对外说明与后台算法的运行规则一致。
- 平台对用户的发放、补偿等行为，一律通过**明示的客服补偿等公开流程**进行，用户可见、可查、留痕，不会伪装成回馈结果。
- 平台保留依法对活动规则进行调整的权利，调整前将提前公示；调整不影响用户已获得的回馈好礼。

### 7.6 回馈资格与消费挂钩

为保障您的知情权，平台特此明确本活动的一项重要机制：

**（一）** 本平台为"消费获取积分、积分参与回馈"的模式。除消耗积分外，您**能够获得哪些价值档位的好礼，与您在平台的累计消费情况相关联**。

**（二）** 平台为每个回馈活动设有好礼预算机制，您的消费会按规则积累相应的回馈资格额度。**当该额度不足时，较高价值档位（高档/中档）的好礼可能暂时无法获得**，您将获得低档位或保底好礼。随着消费的增加，您可解锁更高价值档位的回馈资格。

**（三）** 简而言之：**消费越多，可解锁的好礼档位越高、获得较高价值好礼的机会越大**。这是平台的正当激励机制，而非对特定用户的暗箱操纵。

**（四）** 上述资格机制的内部计算细节（如具体额度数值、算法参数等）属于平台运营信息，不在活动页面逐一展示；但其"消费影响可得档位"的基本规则已通过本条向您明示。

### 7.7 成长等级与回馈档位

**（一）** 平台设有**成长等级体系**（如青铜、白银、黄金、钻石等），您的成长等级由累计消费积分实时派生。

**（二）** 不同成长等级在回馈活动中可享有**不同的较高价值档位获得机会加成**（作为会员权益公示）。该加成依统一规则运行，对同一等级的所有用户一视同仁，不针对特定个人。

**（三）** 各等级的具体加成以活动页面或会员权益页面的实际公示为准。此为**公开的会员权益**。

### 7.8 记录留存

平台依《规范促销行为暂行规定》等要求，如实、准确、完整地记录活动设奖规则、公示信息、发放结果等内容，并妥善保存不少于法定期限，依法接受监督检查。
