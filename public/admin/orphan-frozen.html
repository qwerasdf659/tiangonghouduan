后端API字段说明
检测API (GET /api/v4/console/orphan-frozen/detect)
  • total - 孤儿冻结记录数
  • total_amount - 孤儿冻结总金额
  • orphan_list[] - 孤儿冻结列表
孤儿冻结项字段 (orphan_list item)
  • user_id - 用户ID
  • account_id - 账户ID
  • asset_code - 资产代码
  • frozen_amount - 账户总冻结金额
  • listed_amount - 活跃挂牌冻结金额
  • orphan_amount - 孤儿冻结金额 (frozen_amount - listed_amount)
  • available_amount - 可用余额
  • description - 描述信息
统计API (GET /api/v4/console/orphan-frozen/stats)
  • total_orphan_count - 孤儿冻结总数
  • total_orphan_amount - 孤儿冻结总金额
  • affected_user_count - 影响用户数
  • by_asset[] - 按资产类型分组统计
确认清理孤儿冻结
清理操作会将孤儿冻结金额解冻到用户的可用余额，此操作会记录审计日志。
